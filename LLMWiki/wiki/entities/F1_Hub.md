# F1 Hub

The [[F1_Hub]] is a primary section of the [[Manjanium_On_Softs]] platform focused on racing telemetry. 

## Key Components
- Live Timing Tower
- Race Tracker
- Pit Wall Simulation
- Interactive Track Guides

## Presets
Users can switch between three preset layouts configured via a global settings manager:
1. **Live Focused**: Timing Tower + Globe.
2. **Stats Detailed**: Standings, Telemetry, Weather, and Race Log.
3. **Compact Overview**: Balanced Session, Globe, and Top 10 data.

The hub utilizes the [[Stitch_Design_System]] for a clean, spacious look.
