# Manjanium OS Terminal

The [[Manjanium_OS_Terminal]] is a chat and CLI interface deeply integrated into [[Manjanium_On_Softs]].

## Details
- Implemented as a sleek, glassmorphic vertical sidebar.
- Uses React Portal to break out of CSS transforms for a permanent overlay.
- Powered by Clerk authentication and Supabase real-time synchronization on the `terminal_chats` table.
- Features slash commands like `/predict`, `/stats`, and `/compare`.
