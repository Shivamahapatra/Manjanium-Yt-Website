# Stitch Design System

The [[Stitch_Design_System]] is the central aesthetic architecture for [[Manjanium_On_Softs]]. 

## Style Guide
- **Colors & Tokens**: Defined via CSS variables (e.g., `--f1-accent`, `--f1-surface`, `--f1-border`).
- **Aesthetic**: Premium telemetry styling featuring glassmorphism, background blurs, and precise borders (e.g., 50% opacity borders).
- **Implementation**: Utilizes Tailwind CSS v4 alongside wrapper components like `F1Card`, `F1Badge`, and `FootballCard` to enforce consistency across both the [[F1_Hub]] and the [[Football_Center]].
